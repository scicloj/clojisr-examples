Setup

Imports from clojisr library.

(require
   '[clojisr.v1.r :as r :refer
     [r r->clj clj->r r+ colon bra
      bra<-]]
   '[clojisr.v1.require :refer
     [require-r]]
   '[clojisr.v1.applications.plotting
     :refer [plot->file]]
   '[tech.ml.dataset :as dataset])

Import basic R packages.

(require-r '[base :as base :refer
              [$ <- $<-]]
            '[utils :as u]
            '[stats :as stats]
            '[graphics :as g]
            '[datasets :refer :all])

We will also use def-r macro (not shown here) which helps to define the same things on both sides (Clojure and R).

Ensure R working folder is the same as Clojure.

(base/setwd (.getAbsolutePath
               (java.io.File. ".")))

Options

(base/options :width 120 :digits 7)

Various ways to call R backend.
